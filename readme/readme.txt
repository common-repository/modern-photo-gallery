== Description ==
Modern Photo Gallery Plugin is very simple plugin that allow you to create categorized photo gallery.
You just have to choose photo from the media and add them to your gallery by selecting related category to that photo.

== Installation ==

How to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Photo Gallery menu to configure the plugin
4. Use this `[Modern_Photo_Gallery_ShortCode]` Short Code to display photo gallery any where in the post or in any page.
